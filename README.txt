Bio Lab Insect Explorer

Folder structure:
- index.html
- quiz.html
- css/styles.css
- js/app.js
- images/insects/*.png
- audio/sound/
- audio/narrative/
- audio/quiz/

Audio file names:
Use the insect slug names below.

butterfly.mp3
ant.mp3
grasshopper.mp3
beetle.mp3
fly.mp3
mosquito.mp3
termite.mp3
bee.mp3
wasp.mp3
dragonfly.mp3
stick_insect.mp3
cockroach.mp3

You can also use .wav or .ogg.

Behavior:
- Click/tap an insect card -> plays insect sound from audio/sound/
- Press and hold 2 seconds -> plays insect narration from audio/narrative/
- Only one sound plays at a time
- If no audio files exist yet, the site falls back to generated sound and browser narration

Hosting:
Upload the whole folder to any static host or university web server.
Do not upload only index.html by itself.
